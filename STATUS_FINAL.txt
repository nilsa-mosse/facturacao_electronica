  OPÇÃO 2 - Manual:
  
    1️⃣  Feche o Eclipse completamente
    2️⃣  Abra CMD na pasta: D:\eclipse_workspace
    3️⃣  Execute: rmdir /s /q .metadata\plugins\org.eclipse.jdt.core
    4️⃣  Execute: rmdir /s /q .metadata\plugins\org.eclipse.core.resources
    5️⃣  Abra CMD na pasta: D:\eclipse_workspace\facturacao_electronica
    6️⃣  Execute: mvn clean install -DskipTests
    7️⃣  Reabra o Eclipse
    
    ⏱️  Tempo total: 3-5 minutos


  OPÇÃO 3 - Update Project (no Eclipse):
  
    1️⃣  Right-click no projeto → Maven → Update Project
    2️⃣  Marque "Force Update of Snapshots/Releases"
    3️⃣  Clique OK
    4️⃣  Aguarde rebuild
    
    ⏱️  Tempo total: 1-2 minutos
    ⚠️  Nota: Pode não funcionar se cache estiver muito persistente
